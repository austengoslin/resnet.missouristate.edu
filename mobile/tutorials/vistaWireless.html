Windows Vista Wireless Setup

  1. To start configuring your wireless, click Start Â» Connect To.
  2. A box should open displaying all available wireless networks. Select MSU Resnet and then Connect.
  3. Select Enter/select additional log on information.
  4. To authenticate, enter the following information:
    User Name: BearPass Login
    Password: Missouri State Password
    Logon domain: SGF.
  5. Select Enter/select additional log on information again.
  6. Select OK to accept the server certificate.
  7. You should now be connected to the MSU Resnet network. If this method does not work try connecting manually.
