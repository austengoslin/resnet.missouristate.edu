Finding Your MAC Address

    On Windows Xp

  1. Select "Start" » "Run".
  2. Type cmd into the window and click "OK".
  3. In the newly opened window, type ipconfig /all and press enter.
  4. Displayed will be one or more types of connections. The MAC address will be listed as the physical address.
  5. "Ethernet Adapter Wireless Network Connection" = (Wireless Network Card)
    "Ethernet Adapter Local Area Connection" = (Wired Network Card)

    6. On Windows Vista or 7

  6. Select "Start" and type cmd into the search menu
  7. In the newly opened window, type ipconfig /all and press enter.
  8. Displayed will be one or more types of connections. The MAC address will be listed as the physical address.
  9. "Ethernet Adapter Wireless Network Connection" = (Wireless Network Card)
    "Ethernet Adapter Local Area Connection" = (Wired Network Card)

    10. On Mac OS X

  10. Click the Apple logo » "About this Mac".
  11. Select "Network" from the panel on the left, and then "AirPort" on the top-right panel. The MAC address will be listed in the bottom-right panel.
