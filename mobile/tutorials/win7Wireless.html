Windows 7 Wireless Setup

  1. To start configuring your wireless, click on the network icon and it will show the list of available wireless networks. If you don't see this ensure that your wireless card is on or enabled.
  2. Click on "resnet 40" then click "Connect"
  3. In the next windows enter your BearPass Login and password (what you would use to log into My.MissouriState.edu) and click "OK"
  4. Click "Connect" when the Windows Security Alert pops up.
  5. Windows will finish connecting to the network.
  6. You are now connected to MSU Resnet
