Chrome OS Wireless Setup

  1. Tap Settings.
  2. Tap Wifi.
  3. Tap MSU Resnet from the Wi-Fi Networks screen.
  4. Enter the following credentials:
    UserName:BearPass Login
    Password:Missouri State Password
  5. Tap Accept to accept the server certificate.
  6. You are now connected to MSU Resnet
