Android Wireless Setup

  1. Go into your settings and make sure your Wi-Fi is on and tap it
  2. Select you network from the list. For students living in the residence halls, this will be "MSU ResNet". For students living off campus, this will be "MSU".
  3. The EAP method should be set to PEAP. All other settings can remain as they are.
  4. Use the following credentials:
    Identity: BearPass Login
    Password: Missouri State Password
    Tap Connect
